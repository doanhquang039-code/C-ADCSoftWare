## 🚀 Quick Start

### 1. Clone & Setup
```bash
git clone <repository>
cd WEBDULICH
```

### 2. Database Setup
```bash
# Update connection string in appsettings.json
# Run migrations
dotnet ef database update
```

### 3. Configuration
```bash
# Copy appsettings.example.json to appsettings.json
# Update API keys and secrets
```

### 4. Run Application
```bash
dotnet run
```

